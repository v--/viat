# Viat examples

This directory contains some more thorough usage examples that guided the development of Viat.

* My [dotfiles management](./dotfiles) [fish](https://fishshell.com/) script that uses the command-line `shell-export` API.

* My [bibliography management](./bibliography) Python script that uses the programmatic API.

* [Alrin](https://github.com/v--/alrin), a tool I use to manage [my](https://ivasilev.net/pacman) [pacman](https://pacman.archlinux.page/)/[ALPM](https://alpm.archlinux.page/) repository.
