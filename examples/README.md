# Viat examples

This directory contains some more thorough usage examples that guided the development of Viat.

> [!NOTE]
> The [`viat`](https://pypi.org/project/viat/) package on PyPI includes both the command-line tool and the programmatic API used in the examples.

> [!TIP]
> See the [usage tutorial](https://viat.readthedocs.io/en/stable/usage/) for an outline of how Viat works.

I have added the following examples:

* My [dotfiles management](./dotfiles) [fish](https://fishshell.com/) script that uses the command-line `shell-export` API.

* My [bibliography management](./bibliography) Python script that uses the programmatic API.
